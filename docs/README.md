# docs / screenshots

Put README images here:

- `dashboard.png` — the main dashboard (body map + Highest-ROI moves)
- `item-detail.png` — an item's detail view (e.g. Gastroscopy), showing the
  first-principles / ROI / risk content

Capture them from the **demo data** (never your real data):

1. `npm run setup && npm run dev` → open http://localhost:3000
2. Switch to English with the language toggle (top-right) if you like.
3. Screenshot the dashboard → save as `docs/dashboard.png`.
4. Click an item (e.g. Gastroscopy) → screenshot → save as `docs/item-detail.png`.

On macOS: `Cmd+Shift+4` then drag, or `Cmd+Shift+4` then Space to grab the window.
